Tab Cluster AI gruppiert offene Chrome-Tabs automatisch — wahlweise mit On-Device-KI (Gemini Nano) oder der Gemini API. Open-Source-Alternative zum eingestellten Tab Organizer. Du siehst jeden Vorschlag in der Vorschau, bevor irgendetwas an deinen Tabs verändert wird.

═══════════════════════════════
✦ DREI ANALYSE-MODI
═══════════════════════════════

• On-Device-KI (Gemini Nano)
  Läuft komplett lokal in Chrome über die Prompt API. Kein API-Schlüssel, keine Datenübertragung. Benötigt Chrome 138+ und einen einmaligen Modell-Download von ca. 22 GB.

• Gemini API
  Du verwendest deinen eigenen kostenlosen Schlüssel aus Google AI Studio. Funktioniert auch auf Geräten, die das On-Device-Modell nicht ausführen können. Mehrere Modelle wählbar (Gemini 3.1 Flash-Lite, 3.5 Flash, 2.5 Pro …).

• Nach Domain ordnen (ohne KI)
  Regelbasierte Gruppierung nach Hostname — kein Modell, kein Schlüssel, kein Netzwerk.

═══════════════════════════════
✦ HIGHLIGHTS
═══════════════════════════════

✓ Vorschau vor dem Anwenden — Gruppennamen, Farben und enthaltene Tabs prüfen. Jederzeit abbrechen.

✓ Zusammenführung mit bestehenden Gruppen — vorhandene Gruppen werden erweitert, statt Duplikate zu erzeugen.

✓ Eigene Gruppierungsregeln — freier Text wie „Arbeit von Privat trennen" wird lokal gespeichert und von der KI berücksichtigt.

✓ Eingebaute Diagnose — wenn die On-Device-KI nicht verfügbar ist, zeigt das Popup Chrome-Version, RAM, Flags und Modellstatus mit konkreten Hinweisen.

✓ Mehrsprachige Oberfläche — Deutsch, English, 日本語, Español, Français. UI folgt der Chrome-Sprache, Gruppennamen folgen der Browser-Inhaltssprache (bewusst unabhängig).

✓ Manifest V3, kein Analytics, keine Telemetrie, kein Remote-Code. Quellcode auf GitHub.

═══════════════════════════════
✦ DATENSCHUTZ
═══════════════════════════════

• On-Device: Titel und URLs der Tabs werden lokal in Chrome verarbeitet. Keine Übertragung.

• Gemini API: Titel und URLs gehen mit DEINEM Schlüssel an Googles Generative Language API. Der Entwickler sieht sie nie.

• Nach Domain ordnen: Keine Übertragung.

• API-Schlüssel und Einstellungen liegen ausschließlich in chrome.storage.local dieses Profils. Beim Deinstallieren werden sie gelöscht.

═══════════════════════════════
✦ VORAUSSETZUNGEN
═══════════════════════════════

On-Device-Modus
• Chrome 138 oder neuer
• 16 GB+ RAM oder GPU mit 4 GB+ VRAM
• 22 GB+ freier Speicher (einmaliger Modell-Download)
• Nicht-getaktetes Netzwerk beim ersten Download

Gemini-API-Modus
• Beliebiges Manifest-V3-Chrome
• Kostenloser API-Schlüssel aus Google AI Studio (https://aistudio.google.com/apikey)

═══════════════════════════════
✦ LIMITS
═══════════════════════════════

• Maximal 40 Tabs pro Analyse
• Mindestens 2 ungruppierte Tabs für eine Gruppe
• Angepinnte und chrome://-Tabs werden ausgeschlossen
• Gruppennamen auf 20 Zeichen begrenzt
