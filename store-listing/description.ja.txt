Tab Cluster AI は、開いている Chrome のタブをオンデバイス AI（Gemini Nano）または Gemini API で自動的にグループ化する拡張機能です。Google が廃止した Tab Organizer のオープンソース代替で、AI の提案は必ずプレビューしてから適用できます。

═══════════════════════════════
✦ 3 つの分析方法
═══════════════════════════════

• オンデバイス AI（Gemini Nano）
  Chrome の Prompt API で完全ローカル処理。API キー不要、データ送信なし。Chrome 138 以上と初回の約 22 GB モデルダウンロードが必要です。

• Gemini API
  Google AI Studio で取得した無料 API キーを使用。オンデバイス AI が使えない端末でも動作します。Gemini 3.1 Flash-Lite / 3.5 Flash / 2.5 Flash / 2.5 Pro など複数モデルから選択可能。

• ドメインで整理（AI なし）
  ホスト名でまとめるルールベース方式。モデル・API キー・通信すべて不要。

═══════════════════════════════
✦ 主な特長
═══════════════════════════════

✓ 適用前にプレビュー — 提案されたグループ名・色・含まれるタブを確認してから適用。いつでもキャンセル可能。

✓ 既存グループへ統合 — 既に「仕事」「買い物」などのグループがあれば、新しいタブはそこに統合され、重複しません。

✓ 自分のルールを指定 — 「仕事と私用を分ける」「買い物はドメインごと」などの自由記述メモを保存。AI がそれに従います。ローカル保存のみ。

✓ 詳細な診断情報 — オンデバイス AI が使えない場合、Chrome バージョン・メモリ・flag・モデル状態など原因と対処を表示。

✓ 多言語 UI — English / 日本語 / Deutsch / Español / Français。UI は Chrome の言語、グループ名はブラウザのコンテンツ言語に追従（独立して動作）。

✓ Manifest V3、解析・テレメトリ・リモートコード一切なし。ソースは GitHub で公開。

═══════════════════════════════
✦ プライバシー
═══════════════════════════════

• オンデバイスモード — タブのタイトルと URL は Chrome 内のみで処理。外部送信なし。

• Gemini API モード — タブのタイトルと URL は、あなたの API キーで Google の Generative Language API に送信。作者が見ることはありません。

• ドメインで整理 — 外部送信なし。

• API キー・モデル選択・グループ化方針は、この端末の chrome.storage.local にのみ保存。アンインストールで削除されます。

詳細: https://0xmokuren.github.io/TabClusterAI/PRIVACY.html

═══════════════════════════════
✦ 必要条件
═══════════════════════════════

オンデバイスモード
• Chrome 138 以上
• 16 GB 以上の RAM、または 4 GB 超の VRAM を持つ GPU
• 22 GB 以上の空きディスク（初回モデル DL 用）
• 非従量制ネットワーク（初回 DL 時）

Gemini API モード
• Manifest V3 対応の Chrome
• Google AI Studio の無料 API キー（https://aistudio.google.com/apikey）
• generativelanguage.googleapis.com への到達

═══════════════════════════════
✦ 制限
═══════════════════════════════

• 1 回の分析は最大 40 タブまで（モデルのペイロード上限）
• グループ化には未グループ化タブが 2 つ以上必要（Chrome の仕様）
• ピン留めタブ・chrome:// は対象外
• グループ名は 20 文字まで
